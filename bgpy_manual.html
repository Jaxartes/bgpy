What "bgpy" is

"bgpy" is an implementation of parts of the BGP protocol in Python. It's not a real router but it can send and receive BGP messages and is usable for test purposes.

"bgpy," including this manual, is freely redistributable under a BSD license. The latest version may be found in the GIT repository at https://github.com/Jaxartes/bgpy/.

The name "bgpy" is derived from the name of the protocol (BGP) and the language of its implementation (Python). It is pronounced "bee-gee-pie."
